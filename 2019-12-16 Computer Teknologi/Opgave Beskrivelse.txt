-- Spil hub hjemmeside --
En console app game hub (C#) med ekstern chat (HTML/CSS).
Spillene kører igennem console og der skal være knap til at åbne et chat rum -
til både fælles og privat beskeder.
Der skal være en highscore implemteret i spillene.
Der skal være mulighed for at dele highscores på social media via links.

Fokus bliver på HTML/CSS og Netværk.
Da det er de fag som jeg ikke stærk i.


Fag som indgår:
Clientside
Grundlæggende Prog.
Netværk

Milepæle :
- Spil og Hjemmeside kan snakke sammen
- Chat har fælles og privat chat

Mål:
Mandag : Håber på at være færdig med spil delen så jeg kan begynde på HTML imorgen
Tirsdag : Mangler et enkelt spil i menuen minimum og så skal jeg til HTML og CSS
Onsdag : Implementere Javascript på HTML side for at simulerer en chat uden backend
Torsdag : Lave point system og færdiggøre mindre hængepartier
Fredag : Testing og Aflevering

Afsluttende noter:
- På baggrund af manglende backend var det for svært at få chatten til at fungere som jeg ønskede
- HTML løsningen synes jeg selv er gankse fornuftig selvom det er en sub-optimal løsning.
- All in all har clientside været det sværeste hvilket jeg også vidste det ville blive.
- Den grundlæggende programmerings del var rimelig ligetil indtil internettet forsvandt og dokumentation var utilgængelig.
- Netværksdelen nået jeg aldrig til da HTML / CSS / JS tog den tid det tog.
	- ideen med netværk var at få en client/server forbindelse ->
	  til ikke bare at skrive i console men at add ny HTML på chat siden
- Jeg valgte at fokusere på funktionellitet frem for at tilføje en masse nye features ->
  som måske ville fucke det hele op.
